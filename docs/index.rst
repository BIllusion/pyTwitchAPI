Python Twitch API
=====================================

This is a full implementation of the Twitch Helix API and its Webhook in python 3.7.

On Github: https://github.com/Teekeks/pyTwitchAPI

On PyPi: https://pypi.org/project/twitchAPI/


Installation
============

Install using pip:

```pip install twitchAPI```
